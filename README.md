
# Amazon Frontend Clone

A frontend clone of the Amazon homepage built using pure HTML and CSS.  
This project was created to practice layout design UI structure and basic frontend development concepts.

# Features

- Navigation bar similar to Amazon
- Search bar UI
- Product category sectionscoo
- Background images and banners
- Clean and structured layout using CSS

# Technologies Used

- HTML5
- CSS3
